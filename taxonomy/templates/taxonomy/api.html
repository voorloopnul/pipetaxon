API Documentation

Endpoints

GET /api/taxonomy/

Return the full list of taxonomies in database

GET parameters:

  • rank
  • division
  • search
  • parent

Try it!

GET /api/taxonomy/<$taxid>/

Return taxonomy information for a given taxid.

No extra parameters!

Try it!

GET /api/taxonomy/lca/

Return the lowest common ancestor taxonomy entry for a list of taxid

GET parameters:

  • taxid_list

Try it!

GET /api/taxonomy/<$taxId>/lineage/

Return the entire lineage for a given taxid

No extra parameters!

Try it!

GET /api/taxonomy/accession/<$accessionId>/

Return a taxonomy object from a given sequence accession

No extra parameters!
